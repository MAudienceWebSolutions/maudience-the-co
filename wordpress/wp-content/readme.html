The Company WordPress Theme Read Me

This document describes how to install and use your WordPress theme.

Summary

  1. Plugins

    1. Activate the "Gravity Forms" plugin

      Go to Settings » Gravity Forms and setup plugin options.

      To create a form, click Forms > New Form, and fill form title. After that you can add fields from the right panel, to the left empty panel. When finished click Update Form.

      To add a form, visit any page, and on top of the main WYSIWYG editor you will find Add Form button. Click it, and select the form with the correct settings from the popup. When finished click Insert Form button, and you are done.

    2. Activate the "Post Types Order" plugin

      Go to Settings » Post Types Order and setup plugin options: Minimum Level to use this plugin to Administrator, Auto Sort to disabled and Admin Sort to enabled.

    3. Activate the "Category Order and Taxonomy Terms Order" plugin

      Go to Settings » Taxonomy Terms Order and setup plugin options: Minimum Level to use this plugin to Administrator, Auto Sort to disabled and Admin Sort to enabled.

    4. Activate the "WooCommerce" plugin

      Go to WooCommerce » and setup plugin options

    5. Activate the "VarkTech Wholesale Pricing for WooCommerce" plugin

      Go to Wholesale Pricing Rules » and setup plugin options

  2. Content Management

    1. Page Templates

      • Default Template

        This is the template that will be used by all pages by default.

      • Home

        The home page is divided into 4 sections.

        The first section contains home intro.

        The second section contains home services.

        The third section contains home features.

        The last section contains home logos.

    2. Theme Options

      Go to Theme Options » General and setup header and footer scripts

      Go to Theme Options » Shop and setup single product information for single product pages

      Go to Theme Options » Socials Icons and setup social icons and links

      Go to Theme Options » Contact Info and setup contact info for theme footer

      Go to Theme Options » Contact Hours and setup Contact hours for theme footer

      Go to Theme Options » Copyright and setup Copyright text for theme footer

    3. Sidebar

      Each page can have a unique sidebar associated with it, or a couple of pages could share a common sidebar. There are also 2 available sidebars for the WooCommerce pages - "Shop Sidebar" and "Single Product Sidebar". In order to determine which sidebar should be visible on each page, edit the page and choose an existing sidebar, or create a new one, by selecting the corresponding item from the Sidebar dropdown that appears on the right.

      After that go to Appearance » Widgets to set up the widgets that would appear in the sidebar.

    4. Shortcodes

      • [year]

        Use this shortcode to display the current year.
